# MovieDB project

#### by using OMdb

## Set up project at your localhost:3000

1. Clone this repo
2. Then

```
npm install
```

3. Add API

## Create API

1. Create account at [omdbapi](http://www.omdbapi.com/) then create your own API
2. Create .env file at root (not source)
3. In .env, add REACT_APP_MOVIE_API_KEY={yourAPI}
